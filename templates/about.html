About

This is a simple web page displaying the papers in my BibTeX database. These are curated and in a consistent format.

  1. When arXiv versions exist for a paper, they are linked to.
  2. I actually care about the version of the paper, so we store arXiv versions in separate entries.
  3. When a paper has a DOI, it is linked to.
  4. For every paper that I can cite, I want to be able to easily find the PDF file on my computer. To that end, I store the SHA-256 checksum of the PDF file in the BibTeX. When several versions are available, I store all of the checksums in order of preference.
